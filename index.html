蝦家班 Blueprint Engine V3

痛苦圖紙一鍵出貨中心 — 基於 Project IDX 的零成本 AI 代理工作區自動配置

1. 設定與客製
🧠 工兵大隊 (Sapper Brigade)
預載 Node.js 22 + Docker + 9router AI 閘道，並自動全局配置與安裝 Claude Code CLI。
🦀 龍蝦小兵 (Lobster Soldier)
工兵大隊完整環境 + 自動 Clone/配置與安裝 ClawTeam-OpenClaw 代理框架。
🍃 清爽大軍 (Fresh Army)
極簡環境。僅拉起 Tailscale、Rootless Docker 和 9router 閘道（不預裝 Claude Code/OpenClaw）。
2. 匯出成果
點擊上方按鈕可直接測試在 Project IDX (Firebase Studio) 中建立工作區


        
      

    


    
    

      
🛠️ 這是如何運作的？

      

        

          
1

          
建立自訂範本

          
本專案根目錄已配置 idx-template.json 與 idx-template.nix 作為 Project IDX 官方支援的自訂範本規格。

        

        

          
2

          
使用者點擊連結

          
使用者點選按鈕時，IDX 會解析範本並在網頁中跳出選擇清單，由使用者動態挑選「清爽大軍、工兵大隊或龍蝦小兵」。

        

        

          
3

          
自動建置完成

          
Nix 會執行 bootstrap 腳本，拉下指定部隊組合所屬的環境配置，秒級部署出安全隔離的 Docker & AI 代理環境！
